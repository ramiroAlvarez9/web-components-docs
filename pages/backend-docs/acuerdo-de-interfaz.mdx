## Create new room 

(user click on button "create new room")

### Request

```js

[ POST ] https://localhost:3000/create

```

### Post:

```js

        #body
        {
                "roomName": "roomName",
                "userName": "userName"
        }

```

### Response

```js

        #body
        {

                "response" : {room, user}
                "error": null
        }

```

### Error

```js

        #body
        {
                "error": {status, code}, 
                "response": null   
        }

```
